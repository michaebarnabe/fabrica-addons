# Add-on: Gerador de QR Code SVG

Este add-on web cria QR Codes de alta qualidade em formato SVG a partir de qualquer URL. É ideal para links de sites, perfis de redes sociais, links de WhatsApp e qualquer outra aplicação onde a qualidade visual e a flexibilidade são importantes.

## Funcionalidades

-   Gera QR Codes que não expiram a partir de uma URL.
-   **Saída em SVG:** Produz imagens vetoriais perfeitas, que podem ser redimensionadas infinitamente sem perder a qualidade.
-   **Customização Visual:**
    -   Altere a cor principal do QR Code.
    -   Altere a cor do fundo (suporta fundo transparente, basta escolher a cor branca e usar em design).
    -   Ajuste a margem (padding) ao redor do código.
-   **Download Direto:** Baixe o arquivo `.svg` gerado para uso em projetos de design, web ou impressão.

## Como Usar

1.  Abra o arquivo `index.html` em qualquer navegador moderno.
2.  Cole a URL desejada no campo "URL para o QR Code".
3.  Use os controles para personalizar a cor do QR Code, a cor do fundo e a margem.
4.  Clique no botão "Gerar QR Code".
5.  O resultado aparecerá instantaneamente na área de visualização.
6.  Clique em "Baixar SVG" para salvar o arquivo vetorial em seu computador.
