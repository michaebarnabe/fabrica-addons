# Instagram Carousel Splitter

Este add-on é uma ferramenta web simples e poderosa projetada para social medias e criadores de conteúdo.

## O Problema
Criar carrosséis contínuos para o Instagram manualmente é um processo demorado e propenso a erros, especialmente para quem não domina ferramentas de design complexas como Photoshop ou Figma.

## A Solução
O **Instagram Carousel Splitter** permite que você:
1.  Faça o upload de uma única imagem.
2.  Ajuste a posição, o zoom e a rotação da imagem em uma área de visualização.
3.  Escolha o formato do post (Quadrado 1:1, Retrato 4:5) e a quantidade de slides.
4.  Visualize instantaneamente como a imagem será dividida.
5.  Baixe cada slide individualmente ou todos juntos em um único arquivo `.zip`, prontos para postar.

Esta ferramenta funciona 100% no seu navegador, garantindo privacidade e rapidez.
