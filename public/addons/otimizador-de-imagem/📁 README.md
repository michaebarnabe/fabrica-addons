# Otimizador de Imagens Pro

Uma ferramenta web leve e poderosa para otimizar imagens para a internet. Converta suas fotos para os formatos modernos WEBP ou AVIF diretamente no navegador.

## Funcionalidades

- **Formatos de Entrada:** Suporta JPG, PNG e HEIC (fotos de iPhone).
- **Formatos de Saída:**
  - **WEBP:** O padrão moderno compatível com todos os navegadores.
  - **AVIF:** A nova geração de compressão, oferecendo qualidade superior com arquivos ainda menores.
- **Controle Total:** Ajuste a qualidade da compressão com uma barra deslizante.
- **Privacidade Total:** Todo o processamento é feito localmente. Suas fotos nunca saem do seu computador.
- **Organização:** Suporta seleção de pastas (mantendo a estrutura de diretórios) ou seleção de múltiplos arquivos.

## Como Usar

1.  Abra o `index.html` no seu navegador.
2.  Escolha se quer converter para **WEBP** ou **AVIF**.
3.  Ajuste a **Qualidade** (o padrão 80% é um ótimo equilíbrio).
4.  Selecione sua pasta de imagens ou arquivos individuais.
5.  Clique em "Processar Imagens".
6.  Ao final, baixe tudo em um único arquivo ZIP.
