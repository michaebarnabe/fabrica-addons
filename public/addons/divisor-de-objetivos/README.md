# Divisor de Objetivos

Ferramenta simples que ajuda a dividir um grande objetivo em tarefas menores.

Funcionalidades:
- Campo para objetivo principal
- Botão "Gerar estrutura" para criar subtarefas iniciais
- Adicionar e remover subtarefas manualmente
- Salvamento automático no `localStorage`
- Interface em blocos e código modular

Como usar:
1. Abra o addon
2. Escreva seu objetivo principal
3. Clique em "Gerar estrutura" para sugestões iniciais
4. Edite, adicione ou remova subtarefas
5. Tudo é salvo automaticamente no navegador
