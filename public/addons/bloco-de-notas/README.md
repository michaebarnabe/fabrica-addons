# Bloco de Notas

Um bloco de notas web simples e eficiente que permite criar, editar, salvar e deletar notas com armazenamento local persistente.

## Funcionalidades

- ✅ Criar novas notas
- ✅ Editar notas existentes
- ✅ Excluir notas
- ✅ Buscar notas por título ou conteúdo
- ✅ Importar notas de arquivos TXT
- ✅ Armazenamento persistente com IndexedDB
- ✅ Interface simples e intuitiva

## Como Usar

1. Abra o aplicativo
2. Clique em "Novo" para criar uma nova nota
3. Digite um título e conteúdo
4. Clique em "Salvar" para armazenar
5. Use a barra de busca para encontrar notas específicas
6. Selecione uma nota da lista para editá-la
7. Use "Importar TXT" para importar notas de arquivos

## Armazenamento

As notas são armazenadas no IndexedDB do navegador, garantindo persistência mesmo após fechar a aba.
